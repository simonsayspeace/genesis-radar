# Contributing to The Genesis Radar

All contributions welcome — code, hardware designs, research, documentation.

## How To Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

## Areas Needing Help

- Fusion algorithm mathematics
- Signal processing improvements
- 3D print files for enclosure
- Android CSI app improvements
- Testing and validation results

## Reporting Issues

Use GitHub Issues to report bugs or suggest improvements.

## Code Style

- Arduino C++ — comment everything clearly
- Python — PEP8 compliant
- Document all signal processing maths

## Contact

Raise a GitHub issue for any questions.
